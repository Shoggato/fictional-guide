6.3 -APIs and Geospatial Data Visualization

Overview


In this class, you will be introduced to the Geoapify API to obtain information about geographic areas and the GeoViews Python library to create maps in Jupyter Notebook. Using these new tools and data from the U.S. Census, you will create visualizations to capture the socioeconomic trend of banking deserts.


What You'll Learn


By the end of this lesson, you will be able to:


  • Use the Geoapify API to obtain information about geographic areas.
  • Use the Census API wrapper.
  • Explain the concept of rate limits and the importance of creating test cases before running large scripts.
  • Dissect new API documentation.
